
DOMUS PRO

Sistema de ventas a crédito con:

- Clientes
- Inventario
- Tarjeta de cliente
- Gráficos de ventas
- Recordatorio de clientes con deuda
- App instalable en celular (PWA)

Instalar:

1. Instalar Node.js
2. Abrir carpeta backend
3. Ejecutar:

npm install
npm start

4. Abrir frontend/index.html
